## 🛠️ 制作方法

### 方法 1: 使用 Canva (推荐,最简单)

**步骤**:

1. **访问 Canva**:
   - 网址: https://www.canva.com/
   - 注册/登录账号 (免费)

2. **创建自定义尺寸**:
   - 点击"创建设计"
   - 选择"自定义尺寸"
   - 输入: 1024 x 500 像素
   - 点击"创建新设计"

3. **设计内容**:

   **背景**:
   - 点击左侧"元素" → "渐变"
   - 选择蓝紫色渐变
   - 或者使用纯色背景 (#2196F3)

   **添加应用图标**:
   - 点击"上传" → 上传你的应用图标
   - 拖到画布左侧
   - 调整大小 (约 150x150)

   **添加文字**:
   - 点击左侧"文字" → "添加标题"
   - 输入: "SmartScan"
   - 字体: Roboto Bold 或 Montserrat Bold
   - 大小: 80-100
   - 颜色: 白色
   - 位置: 居中偏左

   - 再添加副标题:
   - 输入: "快速、安全、易用的二维码扫描工具"
   - 字体: Roboto Regular
   - 大小: 36-48
   - 颜色: 白色 (透明度 90%)
   - 位置: 标题下方

   **添加装饰元素** (可选):
   - 点击左侧"元素"
   - 搜索 "QR code" 或 "scan"
   - 选择合适的图标
   - 放置在右侧
   - 调整颜色为白色

4. **导出**:
   - 点击右上角"分享" → "下载"
   - 格式: PNG
   - 点击"下载"

5. **保存**:
   - 保存为: `smartscan-feature-graphic.png`

## 📐 设计规范

### 必须遵守的规则

1. **尺寸**: 必须是 1024 x 500 像素
2. **格式**: PNG 或 JPEG
3. **文件大小**: < 1 MB
4. **内容**:
   - ✅ 可以包含: 应用名称、图标、功能图标、装饰元素
   - ❌ 不能包含: 价格、评分、下载按钮、误导性内容

### 设计建议

1. **清晰易读**:
   - 文字大小适中
   - 对比度高
   - 避免过于复杂

2. **品牌一致**:
   - 使用应用的主色调
   - 风格与应用图标一致

3. **留白适当**:
   - 不要塞满整个画面
   - 给元素足够的呼吸空间

4. **移动端优化**:
   - 考虑在小屏幕上的显示效果
   - 文字不要太小
