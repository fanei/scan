## 🚀 快速开始

### 运行应用
```bash
# 进入项目目录
cd /Users/fancw/StudioProjects/scan/smartscan_app

# 查看连接的设备
flutter devices

# 运行应用
flutter run -d <device_id>

# 或直接运行（自动选择设备）
flutter run
```

### 热重载
应用运行后，修改代码并保存：
- 按 `r` - 热重载（保留状态）
- 按 `R` - 热重启（重置状态）
- 按 `q` - 退出
