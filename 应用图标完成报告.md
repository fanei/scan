## 🔧 如何修改图标

### 方法 1：修改 Python 脚本（推荐）

1. **修改颜色**
```python
# 在 generate_icon.py 中找到这些行
color1 = (76, 175, 80)    # #4CAF50 - 起始色
color2 = (46, 125, 50)    # #2E7D32 - 结束色
```

2. **修改布局**
```python
# 调整这些参数
line_width = int(size * 0.04)    # 线条粗细
corner_size = int(size * 0.18)   # 角的大小
margin = int(size * 0.22)        # 边距
dot_radius = int(size * 0.028)   # 点的大小
spacing = int(size * 0.08)       # 点的间距
```

3. **重新生成**
```bash
cd /Users/fancw/StudioProjects/scan/icon_generator
python3 generate_icon.py
```

### 方法 2：使用 HTML 生成器

1. 打开 `generate_icon.html`
2. 在浏览器中修改代码
3. 实时预览效果
4. 下载新图标

### 方法 3：使用设计工具

1. 用 Figma/Sketch/Photoshop 设计
2. 导出 1024x1024 PNG
3. 替换 `assets/icon/icon.png`
4. 重新运行 `flutter_launcher_icons`
