# SmartScan 应用图标设计方案

**日期**：2026年1月29日  
**设计目标**：简洁、现代、易识别

---

## 🎨 设计理念

### 核心元素
1. **QR 码**：应用的核心功能
2. **扫描框**：代表扫描动作
3. **绿色**：与应用主题色一致（Theme Color: Green）
4. **现代感**：Material Design 3 风格

### 设计风格
- **简约**：避免过度复杂
- **扁平**：Material Design 扁平化风格
- **高识别度**：小尺寸下也清晰可辨
- **专业感**：商务和个人用户都适合

---

## 💡 设计方案

### 方案 A：QR 码 + 扫描框（推荐）

**描述**：
- 中心：简化的 QR 码图案（3x3 点阵）
- 外围：扫描框的四个角
- 背景：绿色渐变（浅到深）
- 点阵：白色

**特点**：
- ✅ 一眼看出是扫码应用
- ✅ 现代、简洁
- ✅ 辨识度高
- ✅ 符合 Material Design

**配色**：
- 主色：`#4CAF50`（Material Green 500）
- 深色：`#388E3C`（Material Green 700）
- 白色：`#FFFFFF`

### 方案 B：放大镜 + QR 码

**描述**：
- 中心：QR 码
- 上方：放大镜图标
- 背景：绿色

**特点**：
- ✅ 强调"扫描"概念
- ⚠️ 可能不够现代

### 方案 C：纯 QR 码变体

**描述**：
- 整个图标就是一个风格化的 QR 码
- 中间有 "S" 字母或扫描图标

**特点**：
- ✅ 极简
- ⚠️ 可能过于简单

---

## 🎯 最终选择：方案 A

**原因**：
1. 平衡了识别度和美观度
2. 符合 Material Design 3
3. 绿色主题与应用一致
4. 扫描框让用户立刻理解功能

## 📐 技术规格

### 尺寸要求

#### Android
- **xxxhdpi**: 192x192 px
- **xxhdpi**: 144x144 px
- **xhdpi**: 96x96 px
- **hdpi**: 72x72 px
- **mdpi**: 48x48 px

#### iOS
- **App Store**: 1024x1024 px
- **iPhone (@3x)**: 180x180 px
- **iPhone (@2x)**: 120x120 px
- **iPad Pro**: 167x167 px
- **iPad**: 152x152 px

#### Google Play
- **Feature graphic**: 512x512 px（高分辨率）

### 文件格式
- **Android**: PNG（透明背景可选，建议白色背景）
- **iOS**: PNG（必须方形，系统自动圆角）
- **Google Play**: PNG 或 JPG

## 🖌️ 设计细节

### 图标组成

```
+----------------------------+
|                            |
|   ┏━━━━┓      ┏━━━━┓      |
|   ┃                 ┃      |
|   ┃    ▪ ▪ ▪        ┃      |
|   ┃    ▪ ▪ ▪        ┃      |
|   ┃    ▪ ▪ ▪        ┃      |
|   ┃                 ┃      |
|   ┗━━━━┛      ┗━━━━┛      |
|                            |
+----------------------------+
```

**元素**：
1. **四个角的扫描框**（白色，粗线条）
2. **中心 3x3 点阵**（白色圆点，代表 QR 码）
3. **渐变绿色背景**（从 #4CAF50 到 #388E3C）
4. **圆角矩形**（Android adaptive icon）

我将创建一个 Flutter 程序来生成图标，然后导出为不同尺寸的 PNG。

2. 绘制渐变背景
3. 绘制扫描框四角
