## 🎨 截图制作方法

### 方法 1: 真机截图 (推荐)

**步骤**:

1. **准备设备**:
   - 使用真实 Android 设备
   - 推荐分辨率: 1080x1920 或更高
   - 清空通知栏

2. **准备内容**:
   - 准备好要扫描的二维码
   - 填写好生成二维码的示例内容
   - 创建一些历史记录

3. **截图操作**:
   - Android: 电源键 + 音量下键
   - 或使用 `adb shell screencap`

4. **整理截图**:
   - 重命名文件(如 `01_scan.png`)
   - 检查尺寸和清晰度

**命令行截图** (通过 ADB):
```bash
# 连接设备
adb devices

# 截图并保存到电脑
adb shell screencap -p /sdcard/screenshot.png
adb pull /sdcard/screenshot.png ./01_scan.png
adb shell rm /sdcard/screenshot.png
```
